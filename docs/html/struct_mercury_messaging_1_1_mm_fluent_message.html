Detailed Description

Fluent message builder for MercuryMessaging.

This struct provides a chainable API for constructing and sending messages through the Mercury framework. Uses struct to avoid heap allocations and maintain high performance.

Phase 2 additions: Supports spatial filtering (Within, InDirection, InBounds), type filtering (OfType, WithComponent), and custom predicates (Where).
