Detailed Description

High-performance message pooling using Unity's ObjectPool.

Eliminates per-message allocations in hot paths.

Usage: var msg = MmMessagePool.GetInt(42, MmMethod.MessageInt, metadata); // ... use message ... MmMessagePool.Return(msg); // Called automatically at end of routing
