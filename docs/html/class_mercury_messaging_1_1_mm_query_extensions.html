Detailed Description

Extension methods for querying MercuryMessaging hierarchies.

Part of DSL Phase 2.2: Hierarchy Query DSL.

Provides LINQ-like querying capabilities for traversing and filtering responder hierarchies. Supports both MmRelayNode and MmBaseResponder.

Example usage:

// Query from relay node
var enemies = relay.Query()
.Descendants()
.OfType<EnemyResponder>()
.Active()
.ToList();
// Execute action on all matching responders
relay.Query()
.Children()
.WithTag(MmTag.Tag0)
.Execute(r => r.DoSomething());
// Query from responder (same API!)
myResponder.Query()
.Siblings()
.OfType<UIResponder>()
.FirstOrDefault();
