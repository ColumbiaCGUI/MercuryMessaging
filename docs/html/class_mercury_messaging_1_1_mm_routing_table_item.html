◆ Handler

Action<MmMessage> MercuryMessaging.MmRoutingTableItem.Handler

Optional direct message handler for fast dispatch.

When set, bypasses virtual dispatch through Responder.MmInvoke(). Performance: ~1-4 ticks vs ~8-10 ticks for virtual dispatch.

Usage: routingTableItem.Handler = (msg) => HandleMessage(msg);

Note: Handler is NOT serialized. Set it in code (e.g., Awake/Start).
