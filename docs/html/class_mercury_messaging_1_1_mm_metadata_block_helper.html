Detailed Description

Helper class to easily create common MercuryMessaging MetadataBlocks.

Phase 6 Optimization: Cached instances to avoid allocation on every access. WARNING: These cached instances are READ-ONLY. Do not modify their fields. If you need to modify a MetadataBlock, use the copy constructor: new MmMetadataBlock(MmMetadataBlockHelper.Default)
