Detailed Description

This is a basic-straight derivation of MmTaskInfoCollectionFileLoader for MmTransformationTaskInfo.

Generally, this is needed in order to allow us to specify the type of data used in the Task Collection, while still giving us the ability to drag-n-drop the component in Unity.
