void MercuryMessaging.MmHierarchyExtensions.MmSetParent ( this MmRelayNode node,
MmRelayNode newParent,
bool worldPositionStays = true )
static

Set the parent of this relay node, properly handling routing table registration.

This is the recommended way to reparent relay nodes at runtime. Handles: unregister from old parent, Unity SetParent, register with new parent.

Parameters
nodeThe node to reparent
newParentThe new parent relay node (null to make this a root node)
worldPositionStaysIf true, the world position is maintained. Passed to Transform.SetParent.
