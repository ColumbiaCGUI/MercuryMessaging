Detailed Description

Pool for HashSet<int> used by VisitedNodes in message routing.

Eliminates per-message HashSet allocations in cycle detection.

Usage: var visitedNodes = MmHashSetPool.Get(); // ... use for cycle detection ... MmHashSetPool.Return(visitedNodes);
