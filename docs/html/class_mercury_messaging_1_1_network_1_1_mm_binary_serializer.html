Detailed Description

Efficient binary serializer for MercuryMessaging messages.

Converts MmMessage objects to compact byte arrays for network transport. This replaces the previous object[] serialization which was inefficient and not portable across different networking backends.

Binary Format:

  • Header (15 bytes fixed):
    • Magic: 4 bytes "MMSG"
    • Version: 1 byte
    • MessageType: 2 bytes (short)
    • MmMethod: 2 bytes (short)
    • NetId: 4 bytes (uint)
    • Metadata: 2 bytes (packed filters + tag)
  • Payload: Variable length, type-specific

uint MercuryMessaging.Network.MmBinarySerializer.PackMetadata ( MmMetadataBlock metadata)
staticprivate

Pack metadata block into 3 bytes (stored as uint, only 18 bits used).

Bits 0-3: LevelFilter (4 bits) Bits 4-5: ActiveFilter (2 bits) Bits 6-7: SelectedFilter (2 bits) Bits 8-9: NetworkFilter (2 bits) Bits 10-17: Tag (8 bits - supports Tag0-Tag7, where Tag7 = 0x80)
