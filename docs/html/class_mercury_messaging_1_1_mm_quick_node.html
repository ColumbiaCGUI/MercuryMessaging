Detailed Description

Modification of the standard MmRelayNode that allows skipping of standard checks to allow for faster propagation.

OBSOLETE: Use MmRelayNode with EnabledChecks = MmRoutingChecks.None instead. This provides the same behavior (bypassing all routing checks) with more granular control.
