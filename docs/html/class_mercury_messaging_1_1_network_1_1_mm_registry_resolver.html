Detailed Description

Simple registry-based GameObject resolver.

Uses a dictionary to map network IDs to GameObjects. Suitable for testing and scenarios where manual registration is acceptable.

For production use with networking backends, prefer backend-specific resolvers (PhotonResolver, FishNetResolver, etc.) that use the backend's native ID system.
