Detailed Description

Load from and Save to XML files.

Data should be fully encapsulated in the object passed to the Generic. See TransformationDataHandler.cs for an example of how to build on the class for additional control. Based on code from http://wiki.unity3d.com/index.php?title=Save_and_Load_from_XML
