# XR Configuration

This folder contains all VR/XR platform configuration files and settings.

## Structure

- **Oculus/** - Oculus/Meta platform configuration
- **MetaXR/** - Meta XR SDK configuration  
- **XR/** - Unity XR Plugin Management settings
- **XRI/** - XR Interaction Toolkit settings
- **CompositionLayers/** - Composition layers configuration

## Notes

- All XR-related settings are consolidated here for easy management
- Platform-specific configurations are kept separate for clarity
- Unity automatically generates and updates these files
- Do not manually edit unless necessary
