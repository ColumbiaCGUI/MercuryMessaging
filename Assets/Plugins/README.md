# Third-Party Assets

This folder contains all third-party plugins, libraries, and Asset Store packages.

## Structure

### Plugins/
Third-party plugins and libraries:
- **ALINE/** - Debug drawing library for Unity
- **EasyPerformantOutline/** - Performant outline effect system
- **QuickOutline/** - Quick outline effect
- **MKGlowFree/** - Glow effect system
- **Photon/** - Photon Fusion networking framework
- **Android/** - Android-specific plugins
- **Vuplex/** - WebView plugin for Unity

### AssetStore/
Unity Asset Store purchases:
- **ModularCityProps/** - Modular city building props (formerly MCP)
- **PBR_TrafficLightsEU/** - PBR European traffic lights
- **TrafficLightsSystem/** - Complete traffic light system with editor tools
- **Skybox/** - Skybox assets

### GraphSystem/
Graph visualization and management:
- **NewGraph/** - Base graph framework (from GitHub)
- **MercuryGraph/** - Mercury-specific graph implementation

## Notes

- Demo content for plugins is kept with each plugin for reference
- Asset Store packages maintain their original folder structure
- To update a third-party package, replace its folder contents
- Check each package's license before modification or redistribution
